# Info for quick_sort.c

## Description
A C program that implements the quicksort algorithm to sort an array of integers provided by the user.

## Samples
### Sample 1
Input:
````
5
3 1 4 2 5
````
Output:
````
Enter size of array:
Enter the elements of the array
Original array: 3 1 4 2 5 
Sorted array: 1 2 3 4 5 

````
Return code: 0

### Sample 2
Input:
````
3
9 0 -1
````
Output:
````
Enter size of array:
Enter the elements of the array
Original array: 9 0 -1 
Sorted array: -1 0 9 

````
Return code: 0

### Sample 3
Input:
````
6
6 5 4 3 2 1
````
Output:
````
Enter size of array:
Enter the elements of the array
Original array: 6 5 4 3 2 1 
Sorted array: 1 2 3 4 5 6 

````
Return code: 0

### Sample 4
Input:
````
1
42
````
Output:
````
Enter size of array:
Enter the elements of the array
Original array: 42 
Sorted array: 42 

````
Return code: 0

## Notes
First input is array size, then space-separated integers. Uses last element as pivot.
