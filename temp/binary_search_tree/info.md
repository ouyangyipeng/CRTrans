# Info for binary_search_tree.c

## Description
Interactive binary search tree (BST) manager with insertion, deletion, traversal display, and node counting operations.

## Notes
Menu-driven program; inputs are menu choices followed by values when needed. Note: leafcount() uses global variable lcount but doesn't reset it between calls, and count_node() sums leafcount (which returns void) and interior count incorrectly.
